# Security Policy

## Reporting a Vulnerability

If you discover a security issue in this repository, do not disclose it publicly.

- Email: `hello@ansvar.ai`
- Include impact, reproduction steps, and affected versions.

We acknowledge reports within 48 hours and coordinate remediation.

## Supported Versions

Only the latest published release is supported for security patches.

## Security Controls

- Automated CI security scanning
- Dependency vulnerability monitoring
- Source-of-truth metadata validation
- Drift detection for legal source anchors
